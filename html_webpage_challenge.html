The div tag defines a division or a section in an HTML document. The div tag is used to group block-elements to format them with CSS. The HTML section element represents a generic section of a document, i.e., a thematic grouping of content, typically with a heading.

The footer tag defines a footer for a document or section. A footer element should contain information about its containing element. A footer element typically contains:

  • authorship information
  • copyright information
  • contact information
  • sitemap
  • back to top links
  • related documents
